# barca
A Python script that gives information about FC Barcelona's games.

Tested on Python 3.6.

## Installation
To install barca from source, first clone the repository and then run:
```
$ python setup.py install
```

## Usage
*Recent results*  
```
$ fcb --results
```
or
```
$ fcb -r
```

*Upcoming fixtures*
```
$ fcb --fixtures
```
or
```
$ fcb -f
```
*Help*
```
$ fcb --help
```
